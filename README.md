# Number guessing AI model
Ai model for guessing numbers based on user's drawings.

## How to use:
Download the zip and launch "NumberGuesingAiModel.Ink"

## Main features:
  -3x5 Board for drawing numbers from 0 to 9.\
  -"Guess" button for guessing the current written number.\
  -"Train" button for training the model on the data in "train.txt" file.\
  -Showcase of the guessed number and it's probability.

## Other features:
  -"Test" button for showcasing results on the test sample.\
  -"SaveTr" and "SaveTe" buttons, after pressing and typing a number in the field above, saves the current cell drawing as that number for either training or testing.\
  -"Null" button for clearing some of the model's weights to break the symmetry. Next training becomes much more effective.\
  -Charts for model's accuracy and degree of error during training and testing.\
